# EthioHealth

EthioHealth is a modern health information website built using React, TypeScript, and Vite. The project focuses on clean structure, reusable components, and a simple, user-friendly interface.

# Live Demo
https://ethio-health-beta.vercel.app/

# About The Project

This project presents health-related information in an organized and structured layout. It demonstrates component-based architecture and modern frontend development practices.

Main features include:

- Multi-section navigation (Home, Traditional, Modern, About)
- Clean and structured layout
- Reusable React components
- Organized project structure
- Deployed production build

# Technologies Used

- React
- TypeScript
- Vite
- CSS
